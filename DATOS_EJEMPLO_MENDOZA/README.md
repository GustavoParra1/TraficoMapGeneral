# Datos Ejemplo Mendoza

Esta carpeta contiene datos de ejemplo para la ciudad de **Mendoza** que pueden cargarse manualmente desde el Dashboard de Clientes.

## Archivos Incluidos

- **siniestros.csv** - Registros de siniestros de tránsito (choques, atropellos, vuelcos, etc.)
- **camaras_publicas.csv** - Cámaras de vigilancia públicas/municipales
- **cameras_privadas.csv** - Cámaras de vigilancia privadas de comercios y negocios
- **semaforos.csv** - Ubicación y estado de semáforos
- **robos_automotor.csv** - Registros de robos de vehículos y partes automotrices
- **escuelas_colegios.csv** - Ubicación de establecimientos educativos (zonas escolares)

## Características

- ✅ Datos georeferenciados (coordenadas lat/lng precisas)
- ✅ Barrios reales de Mendoza: Centro, Dorrego, Godoy Cruz, Maipú
- ✅ Calles y avenidas auténticas de la ciudad
- ✅ Estructura compatible con el Dashboard de Clientes
- ✅ Formato CSV listo para importar

## Cómo Usar

1. Descarga los archivos CSV necesarios
2. Ve al Dashboard de Clientes → Cargar Datos (Upload)
3. Sube los archivos correspondientes según el tipo de dato
4. Los datos aparecerán en el mapa filtrados por ciudad = Mendoza

## Coordenadas Base

Centro de Mendoza: **-32.8895, -68.8575**

Rango de cobertura: ±0.03° (aproximadamente 3km de radio)

## Notas

- Los datos son ficticios para propósitos de demostración
- Las coordenadas están basadas en la geografía real de Mendoza
- Puedes usar estos datos para probar filtros, búsquedas y análisis
